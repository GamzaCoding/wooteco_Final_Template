## 우테코 8기 최종 코딩테스트 준비 템플릿

---
### 핵심
1. 깃허브: Fork하기 => Clone하기 (깃허브 desktop) => Gamza 브랜치 만들기 => initial commit 확인(Pull Request 보내지는지도 확인)
2. Application, ApplicationTest 돌아가는지 확인
3. FreeForm으로 Model, View, Controller, Service(main, sub), Repository 흐름 작성
4. FreeForm 플로우 기반으로 차근차근 기능 구현 문서 만들자
5. 테스트 통과를 기준으로 기능구현을 시작
6. 중간에 테스트 돌아가는지 확인 필수!!!!!!!
7. Validation은 그 객체에서 한다(나중에 리팩터링). Util(Parser)은 따로 패키지 구분
